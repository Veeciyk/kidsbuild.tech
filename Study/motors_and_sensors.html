All About Motors! üîÑ

Motors are like muscles for robots. They turn electrical energy into movement. Let's look at two types:

  • DC Motors: These spin around continuously - perfect for wheels!
  • Servo Motors: These can turn to exact positions - great for robot arms!

üåü Fun Fact:

Servo motors are used in real helicopters and airplanes to control the movement of wings and tail! They're super precise and can move to exact positions.

Activity 1: Control a DC Motor! üéÆ

Physical Components:

  • 1x DC Motor
  • 1x Breadboard
  • 1x 9V Battery with clip
  • 1x Switch
  • Jumper wires

Using Wokwi Simulator:

  1. Open Wokwi and create a new Arduino project
  2. Add a DC motor from the components panel
  3. Connect motor positive to Arduino pin 9
  4. Connect motor negative to ground (GND)

Meet the Sensors! üîç

Sensors are like robot senses - they help robots understand their environment. Here are some common types:

  • Light Sensors: Detect brightness and darkness
  • Sound Sensors: Pick up noises and commands
  • Touch Sensors: Feel when something touches them
  • Ultrasonic Sensors: Measure distances using sound waves

Activity 2: Light-Sensing LED! üí°

Physical Build:

  1. Place the light sensor on your breadboard
  2. Connect LED with 220Œ© resistor
  3. Wire everything to power and ground
  4. Test by covering the sensor!

Wokwi Version:

    // Arduino code for light sensor
    void setup() {
      pinMode(A0, INPUT);    // Light sensor
      pinMode(13, OUTPUT);   // LED
    }
    
    void loop() {
      int light = analogRead(A0);
      if (light < 500) {    // Dark
        digitalWrite(13, HIGH);
      } else {              // Bright
        digitalWrite(13, LOW);
      }
      delay(100);
    }

Super Challenge: Obstacle-Avoiding Robot! üöó

What You'll Need:

  • 2x DC Motors (for wheels)
  • 1x Ultrasonic Sensor
  • 1x Arduino Board
  • Battery pack
  • Wheels and chassis

Wokwi Code:

    // Basic obstacle avoidance code
    void setup() {
      pinMode(9, OUTPUT);    // Left motor
      pinMode(10, OUTPUT);   // Right motor
      pinMode(12, OUTPUT);   // Ultrasonic trigger
      pinMode(11, INPUT);    // Ultrasonic echo
    }
    
    void loop() {
      if (getDistance() < 20) {  // Object closer than 20cm
        turn();
      } else {
        moveForward();
      }
      delay(100);
    }
