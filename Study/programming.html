    • Set Pin 13 HIGH

üöÄ Advanced Path (Text Programming)

    void setup() {
      pinMode(13, OUTPUT);  // LED pin
    }
    
    void loop() {
      digitalWrite(13, HIGH);  // LED on
      delay(1000);            // Wait 1 second
      digitalWrite(13, LOW);   // LED off
      delay(1000);            // Wait 1 second
    }

Using Wokwi (No Arduino? No Problem!):

  1. Go to Wokwi.com
  2. Create a new Arduino project
  3. Add an LED component
  4. Copy the code above
  5. Click Run and watch it blink!

Activity 2: Smart Light Sensor! üåû

Make an LED that turns on when it's dark!

    void setup() {
      pinMode(9, OUTPUT);    // LED pin
      pinMode(A0, INPUT);    // Light sensor pin
    }
    
    void loop() {
      int light = analogRead(A0);
      if (light < 500) {    // If it's dark
        digitalWrite(9, HIGH);
      } else {              // If it's bright
        digitalWrite(9, LOW);
      }
      delay(100);
    }

Wokwi Version:

  1. Add a Light Sensor component
  2. Connect to A0
  3. Add LED to pin 9
  4. Copy the code above
  5. Test by clicking the sensor to change its value!

Super Challenge: Dancing Robot! üï∫

Make a servo motor dance to different positions!

    #include 
    Servo myservo;
    
    void setup() {
      myservo.attach(9);
    }
    
    void loop() {
      // Dance moves!
      myservo.write(0);    // Turn left
      delay(1000);
      myservo.write(180);  // Turn right
      delay(1000);
      myservo.write(90);   // Center
      delay(1000);
    }
