Part 1: Building Your Robot Body üèóÔ∏è

DIY Materials Needed:

  • 1 Large cereal box or sturdy cardboard
  • 2 DC motors (can be from old toys!)
  • 2 Wheels (bottle caps work great!)
  • 1 Marble or small ball (for front wheel)
  • Glue gun or strong tape
  • Scissors
  • Rubber bands

Building Steps:

  1. Make the chassis:
    • Cut a 15cm x 20cm rectangle from cardboard
    • Fold 2cm up on all sides to make edges
    • Reinforce corners with extra cardboard
  2. Create motor holders:
    • Cut 2 small cardboard strips (3cm x 8cm)
    • Fold into L-shapes
    • Hot glue or tape to hold motors
  3. Make DIY wheels:
    • Use large bottle caps or cut circles from thick cardboard
    • Add rubber bands around edge for grip
    • Hot glue to motor shafts
  4. Front wheel:
    • Make a small cardboard pocket
    • Place marble inside (it should roll freely)
    • Attach to front of chassis

Wokwi Alternative:

  1. Open Wokwi and create new project
  2. Add these components:
    • 1x Arduino Uno
    • 2x DC Motor
    • 1x L298N Motor Driver
  3. Connect virtually (we'll add code later!)

Part 2: Adding Electronics üîå

Materials Needed:

  • 1x Arduino Uno
  • 2x IR line sensors (need to buy these)
  • 4x AA battery holder
  • Jumper wires (can use old headphone wires!)
  • Small breadboard (or cardboard with pushpins)

DIY Breadboard Alternative:

  1. Cut a small piece of cardboard (10cm x 5cm)
  2. Push pins in rows (like a real breadboard)
  3. Wrap wire ends around pins to connect
  4. Secure with tape

Wiring Steps:

  1. Mount Arduino:
    • Create cardboard platform
    • Secure with rubber bands or tape
  2. Connect motors:
    • Motor A to pins 5 and 4
    • Motor B to pins 3 and 2
    • Use tape to secure connections
  3. Add sensors:
    • Mount 2cm from ground using cardboard stands
    • Connect to A0 and A1
    • Secure wires with tape

Wokwi Version:

  1. Add two IR sensors to your project
  2. Connect virtually:

Part 3: Programming Your Robot üíª

The Code (Works for both DIY and Wokwi!):

    // Simple version of our line follower code
    const int leftMotorPin = 5;
    const int rightMotorPin = 3;
    const int leftSensor = A0;
    const int rightSensor = A1;
    
    void setup() {
      pinMode(leftMotorPin, OUTPUT);
      pinMode(rightMotorPin, OUTPUT);
      Serial.begin(9600);  // For testing
    }
    
    void loop() {
      int leftValue = analogRead(leftSensor);
      int rightValue = analogRead(rightSensor);
      
      // Print values for testing
      Serial.print("Left: ");
      Serial.print(leftValue);
      Serial.print(" Right: ");
      Serial.println(rightValue);
      
      if (leftValue < 500 && rightValue < 500) {
        // Both sensors on white - go forward
        analogWrite(leftMotorPin, 200);
        analogWrite(rightMotorPin, 200);
      }
      else if (leftValue >= 500) {
        // Left sees line - turn right
        analogWrite(leftMotorPin, 200);
        analogWrite(rightMotorPin, 0);
      }
      else if (rightValue >= 500) {
        // Right sees line - turn left
        analogWrite(leftMotorPin, 0);
        analogWrite(rightMotorPin, 200);
      }
      
      delay(100);  // Small delay to prevent jitter
    }

Part 4: Testing Time! üîç

DIY Test Track:

  1. Make your track:
    • Use white paper or cardboard
    • Draw thick black line with marker
    • Start with straight line, then try curves!
  2. Test the sensors:
    • Move black paper under sensors
    • Watch Serial Monitor numbers
    • Adjust sensor height if needed

Wokwi Testing:

  1. Use built-in line sensor simulator
  2. Click sensors to change values
  3. Watch how motors respond
  4. Try different sensor positions

Common Problems and Fixes üîß

  • Robot not moving: Check battery connections and motor tape
  • Moving in circles: Make sure both wheels have good grip
  • Not following line: Adjust sensor height or redraw line thicker
  • Wobbly movement: Add more cardboard support or adjust marble wheel
