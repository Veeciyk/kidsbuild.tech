Activity 1: Blink an LED! üí°

Physical Components:

  • 1x Arduino Uno
  • 1x USB Cable
  • 1x LED
  • 1x 220Œ© Resistor
  • Some jumper wires

Using Wokwi Instead:

  1. Go to Wokwi.com and create a new Arduino project
  2. Add an LED to your project from the components panel
  3. Connect the LED's long leg (positive) to pin 13
  4. Connect the LED's short leg to GND through a resistor
    // This makes the LED blink!
    void setup() {
      pinMode(13, OUTPUT);  // Tell Arduino pin 13 is an output
    }
    
    void loop() {
      digitalWrite(13, HIGH);  // Turn LED on
      delay(1000);            // Wait 1 second
      digitalWrite(13, LOW);   // Turn LED off
      delay(1000);            // Wait 1 second
    }

Activity 2: Make a Traffic Light! üö¶

Physical Components:

  • 1x Arduino Uno
  • 3x LEDs (Red, Yellow, Green)
  • 3x 220Œ© Resistors
  • Jumper wires

Wokwi Version:

    // Traffic Light Code
    void setup() {
      pinMode(11, OUTPUT);  // Red LED
      pinMode(12, OUTPUT);  // Yellow LED
      pinMode(13, OUTPUT);  // Green LED
    }
    
    void loop() {
      // Green light
      digitalWrite(13, HIGH);
      delay(3000);
      digitalWrite(13, LOW);
      
      // Yellow light
      digitalWrite(12, HIGH);
      delay(1000);
      digitalWrite(12, LOW);
      
      // Red light
      digitalWrite(11, HIGH);
      delay(3000);
      digitalWrite(11, LOW);
    }
