Challenge 1: The Amazing Maze! üåü

DIY Maze Setup:

  1. Create your maze:
    • Use cardboard walls or black tape on white paper
    • Make paths about 15cm wide for DIY robots
    • Start simple, then add curves and turns
  2. Scoring:
    • 5 points for each turn completed
    • 10 bonus points for no wall touches
    • Timer bonus: Finish under 1 minute!

Wokwi Version:

    // Enhanced maze navigation code
    const int TURN_DELAY = 750;  // Adjust for DIY wheels
    
    void loop() {
      int leftSensor = analogRead(A0);
      int rightSensor = analogRead(A1);
      
      if (leftSensor < 500 && rightSensor < 500) {
        moveForward();
      } else if (leftSensor >= 500) {
        turnRight();
        delay(TURN_DELAY);
      } else if (rightSensor >= 500) {
        turnLeft();
        delay(TURN_DELAY);
      }
    }

Challenge 2: Speed Racing! üèéÔ∏è

DIY Race Track:

  1. Create the track:
    • Make a large figure-8 with black tape
    • Add checkpoints every quarter lap
    • Optional: Add 'boost zones' for extra fun!
  2. Speed Upgrades:
    • Adjust motor power in code
    • Add fresh batteries
    • Check wheel alignment

Wokwi Race Mode:

    // Speed racing configuration
    void setup() {
      // Increase motor power
      analogWrite(enableA, 200);  // More power!
      analogWrite(enableB, 200);  // Zoom zoom!
    }
    
    void moveForward() {
      // Turbo mode for straights!
      digitalWrite(leftMotorA, HIGH);
      digitalWrite(rightMotorA, HIGH);
      digitalWrite(leftMotorB, LOW);
      digitalWrite(rightMotorB, LOW);
    }

FINAL CHALLENGE: Robot Talent Show! üåü

Prepare Your Robot:

  1. Choose your talent:
    • Dancing Robot (program different moves)
    • Artist Robot (attach a marker)
    • Musical Robot (add a buzzer)
    • Stunt Robot (do tricks)
  2. Customize your robot:
    • Decorate with craft supplies
    • Add LED lights for effects
    • Make a costume from paper

Sample Dance Code:

    // Robot Dance Party!
    void loop() {
      spinMove();
      delay(1000);
      moonwalk();
      delay(1000);
      wiggle();
      delay(1000);
    }
    
    void spinMove() {
      // Spin in place
      digitalWrite(leftMotorA, HIGH);
      digitalWrite(rightMotorB, HIGH);
      delay(500);
    }
    
    void moonwalk() {
      // Back and forth
      moveForward();
      delay(200);
      moveBackward();
      delay(200);
    }
    
    void wiggle() {
      // Quick left-right movements
      turnLeft();
      delay(100);
      turnRight();
      delay(100);
    }
